# Model-based reinforcement learning tools

Tools for the evaluation of model based reinforcement learning solutions.

This package is delivered with the `rl_simulator` repository. To install it run `pip install .` in this directory. Installing this package will give you access to the following commands:
 - `mbrl-run` to run the model-based RL loop (on an env, specifying a model and an agent)
 - `mbrl-simulate` to simulate and save traces using the pickled models learned by `ramp-test` on the static sets.
